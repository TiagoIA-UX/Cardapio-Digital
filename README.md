# Cardápio Digital

Site profissional com **cardápio digital** para restaurantes, lanchonetes, bares e delivery. Landing page, templates responsivos, integração com WhatsApp e **Google Maps** para localização do estabelecimento.

## Funcionalidades

- **Cardápio digital** — site com menu, categorias, fotos e preços
- **WhatsApp** — pedido enviado direto pelo app, sem comissão
- **Google Maps integrado** — link para o cliente localizar o endereço no mapa (formato atual da API Google Maps)
- **Painel** — edição de produtos, categorias e dados do negócio
- **Templates** — Restaurante, Pizzaria, Lanchonete, Bar, Cafeteria, Açaí, Sushi

## Stack

- Next.js 16, React 19, TypeScript
- Supabase (auth, banco)
- Tailwind CSS
- Vercel (hospedagem)

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
