### Perfil Ideal do Parceiro Beta

| Critério                 | Por quê                                                       |
| ------------------------ | ------------------------------------------------------------- |
| Faturamento R$15-50k/mês | Grande o suficiente para ter dor, pequeno para decidir rápido |
| Usa iFood + WhatsApp     | Já conhece delivery digital, sente a dor da taxa              |
| Dono presente            | Decisor na loja, não precisa "consultar sócio"                |
| Bairro classe B          | Ticket médio bom, clientes usam celular                       |
| 1-3 funcionários         | Operação simples, fácil treinar                               |

### Script de Abordagem Presencial

**Horário:** Terça ou Quarta, 15h-17h (pós-almoço, pré-pico)

**Entrada:**

> "Boa tarde! Sou [Nome], trabalho com tecnologia para pizzarias. Vocês usam iFood aqui?"

**Se sim (99% dos casos):**

> "Quanto vocês pagam de taxa por mês? Uns 20%, 25%?"

**Deixar ele falar a dor.**

**Gancho:**

> "Eu criei um sistema que deixa você ter seu próprio cardápio digital, igual iFood, mas sem pagar taxa nenhuma. Tô procurando 2 pizzarias aqui da região pra testar de graça por 30 dias. Posso mostrar em 5 minutos?"

### O que mostrar (5 minutos máximo)

1. **Abrir cardápio de demo no celular DELE** (não no seu)
2. **Montar uma pizza no wizard** - deixar ELE tocar
3. **Mostrar o carrinho e checkout**
4. **Abrir painel de pedidos** - mostrar simplicidade

**Frase de fechamento:**

> "Se você topar, amanhã eu configuro tudo com seu cardápio real. Você só precisa me mandar fotos dos produtos por WhatsApp."

## 3️⃣ ARGUMENTO ANTI-IFOOD

### O Cálculo da Verdade

**Cenário: Pizzaria com 300 pedidos/mês**

| Item              | iFood        | Seu Sistema  |
| ----------------- | ------------ | ------------ |
| Ticket médio      | R$55         | R$55         |
| Faturamento       | R$16.500     | R$16.500     |
| Taxa iFood (27%)  | -R$4.455     | R$0          |
| Mensalidade       | R$0          | -R$99        |
| **Lucro líquido** | **R$12.045** | **R$16.401** |
| **Economia/mês**  | -            | **+R$4.356** |

**Frase de impacto:**

> "Você tá dando R$4.000 por mês pro iFood. Com R$99 você fica com tudo."

### Por que NÃO abandonar iFood (ainda)

| Use iFood para | Use seu sistema para |
| -------------- | -------------------- |
| Novos clientes | Clientes recorrentes |
| Descoberta     | Fidelização          |
| Marketing pago | Canal direto         |

**Estratégia híbrida:**

> "O iFood é seu outdoor digital. Mas todo cliente que já te conhece, você manda pro SEU cardápio. Cada pedido direto é R$15 que fica no seu bolso."

### Script Anti-iFood

> "Me diz uma coisa: quando o cliente pede no iFood, de quem ele é cliente? Dele ou seu?"
>
> [Espera resposta]
>
> "Exato. O iFood não te dá nem o telefone do cliente. Você trabalha, ele fica com o relacionamento E 27% do seu dinheiro."
>
> "Com seu cardápio próprio, o cliente é SEU. Pede direto, você tem WhatsApp dele, pode mandar promoção, cupom. E não paga taxa."
