# REGRA SEO ENTERPRISE - TOP 10

Status: ATIVA
Data: 2026-04-12
Escopo: conteudo de paginas, landing pages, guias e blog (incluindo integracoes externas)

## 1) Intencao de busca primeiro

- Classificar cada pauta como: informacional, comercial, transacional ou navegacional.
- Antes de publicar, comparar com top resultados e entregar melhor profundidade ou foco.

## 2) EEAT obrigatorio

- Experiencia real, expertise, autoridade e confiabilidade.
- Conteudo original, com exemplos reais, dados verificaveis e autor identificado.
- Revisao periodica para manter atualidade.

## 3) Core Web Vitals

- LCP < 2.5s
- CLS baixo
- INP saudavel
- Monitorar com PageSpeed Insights e ajustes continuos.

## 4) Mobile first

- Layout responsivo
- Texto legivel sem zoom
- Alvos de toque confortaveis
- Sem popups agressivos

## 5) On-page estrategico

- Palavra-chave principal em: H1, primeiro paragrafo, URL e meta description.
- Estrutura H2/H3 clara.
- Links internos estrategicos.
- ALT descritivo em imagens.
- Proibido keyword stuffing.

## 6) Backlinks de qualidade

- Priorizar links relevantes (parcerias, PR digital, guest posts qualificados).
- Um link forte vale mais que dezenas de links fracos.

## 7) SEO tecnico impecavel

- sitemap.xml e robots.txt corretos
- HTTPS ativo
- URLs limpas
- Schema Markup
- Correcao de 404 e cadeias de redirecionamento

## 8) Atualizacao constante

- Atualizar paginas ja ranqueadas
- Melhorar dados/estatisticas
- Expandir conteudo com novas perguntas reais

## 9) CTR e engajamento

- Titulo com valor claro
- Meta description persuasiva
- Uso moderado de listas, numeros e promessa objetiva

## 10) SEO para IA e respostas diretas

- Blocos objetivos para featured snippets
- Estrutura em perguntas e respostas
- Conteudo preparado para busca por voz e experiencias generativas

## Regra de integridade de dados

- Proibido inventar estatisticas, depoimentos ou estudos.
- Toda afirmacao de mercado precisa fonte verificavel.
