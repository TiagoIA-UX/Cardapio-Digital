# Licença Comercial de Templates

## Resumo prático

- O código-fonte e os templates continuam protegidos pela BUSL 1.1 e por concessão comercial específica.
- O cliente pagante recebe direito de uso conforme o plano contratado.
- O acesso ao repositório privado não transfere propriedade intelectual.

## O que o cliente pode fazer

- Clonar o repositório privado enquanto mantiver acesso concedido.
- Usar o template contratado no escopo autorizado comercialmente.
- Fazer ajustes internos para operar o próprio delivery dentro do escopo contratado.

## O que o cliente não pode fazer sem contrato adicional

- Revender o código-fonte.
- Redistribuir templates ou trechos para terceiros.
- Publicar fork público.
- Criar SaaS concorrente com o código recebido.
- Compartilhar acesso com outras contas GitHub ou empresas do grupo sem autorização.

## Boas práticas comerciais

- Conceder acesso somente após pagamento confirmado.
- Vincular a licença a um usuário GitHub nominal.
- Manter histórico de concessão e revogação.
- Usar planos e valores documentados no processo comercial.
